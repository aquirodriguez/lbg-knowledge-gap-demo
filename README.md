# LBG Knowledge Gap Flow Prototype

This standalone frontend prototype demonstrates the internal Lux Bond & Green knowledge gap moment: an employee asks a follow-up question, receives the safest approved answer available, and sees the question gracefully captured for review.

## Tech Stack

- Vite
- React
- Tailwind CSS
- Motion for React

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy To Vercel

Import the project into Vercel and use the default Vite settings:

- Build command: `npm run build`
- Output directory: `dist`

## Note

This is frontend-only. It has no real AI calls, backend, database, authentication, trainer portal, admin portal, study guide generator, or real notifications.
